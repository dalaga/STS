## David Alaga - Week 3 Scripts

The script for Week 3 consists of 2 sections:

Script 1 - This contains the basic template for setting a user with basic permissions using the "myuser" using the TYPE:  "AWS::IAM::User" to create the user and basic access policy information

Script 2 - This contains the basic template for setting up a billing alarm "SpendingAlarm:" using the Type: "AWS:CloudWatch::Alarm"

Both Scripts are from the documentation:
Links are as follows:

Billing Alarm: JSON format:
http://docs.aws.amazon.com/AWSCloudFormation/latest/UserGuide/quickref-cloudwatch.html#cloudwatch-sample-billing-alarm

User: JSON format:
https://docs.aws.amazon.com/AWSCloudFormation/latest/UserGuide/quickref-iam.html#scenario-iam-user
